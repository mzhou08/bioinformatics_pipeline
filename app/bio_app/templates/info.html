The pipeline contains a Django web application with the gene/protein sequence data stored in a database. The user can make protein/gene homology and phylogeny analysis requests inside the web application. We use Jenkins scheduler server to arrange multiple Python scripts to run on scheduled time intervals, for tasks such as motif analysis, protein database lookup, multiple sequence analysis, phylogenetic tree generation, etc.
